Collecting Govt. Data:
SAFAR Project

Govt. of India, in collaboration with Indian Institute of Tropical Meteorology, started project SAFAR which measure air quality data in real time but does not have a public API to relase the data. We are monitoring the readings by this project and making trends and data publicly available here.

Publicising the data
through Twitter Feed

We are publicising the data collected from the SAFAR Project through twitter feed! You can follow us and get regular updates on air quality. We are now only in Mumbai but soon expand to Delhi and Pune.

Making Data Public
through set of APIs

We have created few free APIs which can be used by anyone including developers to use this data and develop over it. More details about it can be found here.
